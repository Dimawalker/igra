### Реализация метода generate интерфейса GeneratePreset

Данный метод формирует пресет армии компьютера. Сложность: 

O(n · m), где

n – количество типов юнитов,
m – максимальное количество юнитов в армии.

### Реализация метода simulate интерфейса SimulateBattle.

Данный метод осуществляет симуляцию боя между армией игрока и армией компьютера.

Алгоритмическая сложность:

формирование и сортировка очереди ходов – O(n log n) на раунд;
количество раундов ограничено числом ходов юнитов.
Итоговая сложность:
O(n² log n), где n – общее количество юнитов в обеих армиях.

### Реализация метода getSuitableUnits интерфейса SuitableForAttackUnitsFinder

Метод определяет список юнитов, подходящих для атаки, для атакующего юнита одной из армий. 

**O(n) сложность, правильно учитывает прикрытие
